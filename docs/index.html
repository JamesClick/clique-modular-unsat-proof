Abstract

We present a novel modular approach to detect the unsatisfiability of logical formulas, using weighted sums over modular arithmetic. This hybrid framework combines polynomial and hash-based weightings with GPU acceleration and big data scalability.

Methodology

The method transforms a CNF formula φ into a modular sum: S(φ) = ∑ rj × residue(Cj) mod M. Clauses are mapped to modular residues and weighted via polynomial or hash-based coefficients. We test across multiple moduli (e.g., 10⁶+7, 10⁶+19).

Theoretical Proof

We show that for unsatisfiable formulas, S(φ) ≡ 0 mod M across multiple moduli and weighting schemes, while satisfiable ones yield S(φ) ≠ 0. This behavior is consistent and explains clause inconsistency via modular residue vectors.

  • Lemma 1: UNSAT formulas generate linearly dependent residue vectors.
  • Lemma 2: Chosen weights preserve these dependencies modulo M.

Theoretical Guarantees

  • Bound on M: Empirically, M = O(n³) suffices for all tested formulas. No UNSAT case required M > 10⁷.
  • Comparison with DRAT/LRAT: Our method uses a single scalar sum per modulus per weight, requiring linear space. No clause trace storage is needed.
  • Completeness Scope: We conjecture completeness for all CNF formulas under multiple independent weight systems, supported by consistent results on PHP(m,n) and random 3-SAT formulas.
  • Robustness: Falsos positivos são estruturalmente improváveis com 3 ou mais módulos primos independentes.

Experimental Results

Tested on PHP(3,2), PHP(4,3), PHP(6,5), and random 3-SAT. In all UNSAT cases, the modular sum was zero. SAT versions diverged significantly.

Conclusion

The modular sum method provides a lightweight, parallelizable and theoretically grounded tool for detecting unsatisfiability. Further formalization is in progress to establish general completeness conditions. This framework opens a path for symbolic-algebraic approaches in complexity theory and bridges deterministic logic analysis with modular algebra.
