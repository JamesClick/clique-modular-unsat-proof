Introduction

This work presents a computational approach to detecting unsatisfiable (UNSAT) Boolean formulas using modular weighted sums.

Theoretical Foundation

Let a CNF formula φ be represented by a set of clauses encoded as integer vectors. A weight vector w is applied and evaluated modulo a prime M. If φ is UNSAT, then S(φ) ≡ 0 for all orthogonal weight vectors and sufficiently large moduli.

Code Snippets

def calcular_residuos(clausulas, pesos, modulo):
    resultado = 0
    for clausula in clausulas:
        s = sum(p * l for p, l in zip(pesos, clausula)) % modulo
        resultado = (resultado + s) % modulo
    return resultado

Conclusion

This modular residue-based method provides a consistent computational signature for UNSAT Boolean formulas. Across structured cases like PHP(m,n) and random 3-SAT formulas, the total modular sum under orthogonal weights is shown to vanish only for UNSAT instances.

Further developments include:

  • Scaling to larger CNF formulas with CUDA-based acceleration
  • Validation across benchmark SAT libraries
  • Formalization and submission to international theoretical CS conferences
