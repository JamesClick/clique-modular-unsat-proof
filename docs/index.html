Introduction

This project presents a novel algebraic method for detecting unsatisfiable CNF formulas using modular weighted sums.

Theoretical Foundation

Given a CNF formula \( \phi \), represented as integer vectors, we apply weight vectors \( w \) and evaluate the sum \( S(\phi) = \sum w_i c_i \mod M \). If \( \phi \) is UNSAT, then \( S(\phi) \equiv 0 \) across all orthogonal weights and sufficiently large primes \( M \).

Example

Example: \( \phi = \text{PHP}(5,4),\ w = (2,3,5),\ M = 7 \Rightarrow S(\phi) = 0 \).

Weight Strategies

Exponential, random, and low-discrepancy weights were tested against various SAT and UNSAT formulas.
