Resumo do Projeto

Este projeto apresenta uma abordagem baseada em resíduos modulares para detectar fórmulas booleanas insatisfatíveis (UNSAT). A conjectura é baseada na ideia de que S(φ) ≡ 0 mod M para todas as bases ortogonais de pesos em fórmulas UNSAT.

Conjectura Modular

Seja φ uma fórmula booleana em CNF. Se φ for insatisfatível, então para qualquer vetor de pesos ortogonal e módulo primo suficientemente grande M, temos:

\[ S(\varphi) \equiv 0 \mod M \]

Algoritmo

def calcular_residuos(clausulas, pesos, modulo):
    resultado = 0
    for clausula in clausulas:
        s = sum(p * l for p, l in zip(pesos, clausula)) % modulo
        resultado = (resultado + s) % modulo
    return resultado
