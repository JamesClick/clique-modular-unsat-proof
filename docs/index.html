Abstract Methodology Theoretical Proof Results Graphs Conclusion

Abstract

We present a novel modular approach to detect the unsatisfiability of logical formulas, using weighted sums over modular arithmetic. This hybrid framework combines polynomial and hash-based weightings with GPU acceleration and big data scalability. The method was tested on unsatisfiable PHP(m,n) formulas and random 3-SAT instances.
