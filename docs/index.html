Formal Proof of UNSAT via Modular Weighted Sums

Introduction Method Results Graphs References

Introduction

This project proposes a novel modular method to detect UNSAT CNF formulas using deterministic vector summation techniques. The approach is grounded in linear algebra and validated through CUDA-based simulations.

Method

We calculate weighted modular sums S(φ) using exponential, polynomial and hash-based weights, across distinct prime modules (e.g., 10⁶+7). If S(φ) ≡ 0 for all combinations, φ is UNSAT.

Results

  • Tested PHP(3,2) to PHP(6,5) → S(φ) ≡ 0
  • Random 3-SAT UNSAT formulas → S(φ) ≡ 0
  • Modified SAT versions → S(φ) ≠ 0
