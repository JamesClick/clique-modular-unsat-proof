Formal Proof of UNSAT via Modular Weighted Sums

This project demonstrates a new approach for detecting UNSAT formulas through modular weighted sums. We apply optimized backtracking, GPU parallelization, and mathematical formalization to prove the zero-sum condition S(φ) ≡ 0 mod M for UNSAT CNF formulas.
